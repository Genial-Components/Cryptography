Cryptography component to encrypt sensitive information and one-way hash passwords.

-------
## Prerequisites
- PHP > 7.2.0

## Installation
```sh
cd path/to/this-project
composer require genial-framework/cryptography
```
> Composer is not required but it is recommended.

## Testing
```sh
composer run test
```
